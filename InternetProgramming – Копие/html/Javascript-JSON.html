JSON - JavaScript Object Notation

JSON (JavaScript Object Notation) е лек, текстов формат за обмен на данни, който се използва за предаване на структурирани данни между различни компютърни системи. JSON е базиран на синтаксиса на JavaScript, но е независим от езика и се използва широко в много програмни езици.

Ето някои ключови характеристики на JSON в контекста на JavaScript:

  • Синтаксис: JSON се състои от два основни типа структури - обекти и масиви. Обектите в JSON се описват във вид на ключ-стойностни двойки, разделени със запетаи и оградени с фигурни скоби {}. Масивите в JSON са списъци от стойности, разделени със запетаи и оградени с квадратни скоби []. Например:
    {
    "name": "John",
    "age": 30,
    "city": "New York",
    "hobbies": ["reading", "running", "cooking"],
    "address": {
    "street": "123 Main St",
    "city": "Anytown",
    "zip": "12345"
    }
    }
  • Данни и типове: JSON поддържа различни типове данни, включително низове, числа, булеви стойности, масиви, обекти и null. Той не поддържа специфични типове данни, като функции или обекти с методи.
  • Използване в JavaScript: В JavaScript, JSON се използва за обмяна на данни между клиентския и сървърния код, като се използват методите JSON.parse() и JSON.stringify(). JSON.parse() се използва за преобразуване на JSON низ към обект, докато JSON.stringify() се използва за преобразуване на JavaScript обект или стойност към JSON низ.
    Например, за да преобразувате обект в JSON низ:

    var obj = { name: "John", age: 30 };
    var jsonStr = JSON.stringify(obj);
    console.log(jsonStr); // Очакван изход: {"name":"John","age":30}

    За да преобразувате JSON низ в обект:
    var jsonStr = '{"name":"John","age":30}';
    var obj = JSON.parse(jsonStr);
    console.log(obj.name); // Очакван изход: "John"
  • Поддръжка: JSON е поддържан от почти всички съвременни програмни езици и е популярен формат за обмяна на данни в мрежовите комуникации. Той се използва широко в уеб разработката за комуникация между клиентски и сървърни приложения.

JSON е удобен и ефективен формат за обмяна на данни и се използва широко в много различни приложения. Той е лесен за четене и разбиране както от хора, така и от машини, което го прави предпочитан избор за обмяна на данни във всякакъв вид приложения.
