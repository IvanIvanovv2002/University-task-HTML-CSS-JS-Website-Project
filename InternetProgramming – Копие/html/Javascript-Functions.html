Масиви

Функциите в JavaScript са основен строителен блок на езика. Те позволяват на програмистите да създават блокове от код, които могат да се изпълняват повторно и да се преизползват в различни части на програмата. Ето някои ключови характеристики и аспекти на функциите в JavaScript:

  • Създаване на функции: Функциите в JavaScript могат да бъдат декларирани по няколко начина. Един от тях е чрез ключовата дума function, следвана от името на функцията и списък с параметрите в скоби (), след което тялото на функцията във фигурни скоби {}. Например:
    function greet(name) {
    console.log("Hello, " + name + "!");
    }
  • Извикване на функции: За да извикате функция, използвайте нейното име, последвано от скоби (), които могат да съдържат аргументите, подадени на функцията при извикването й. Например:
    greet("John");
  • Връщане на стойност от функция: Функциите в JavaScript могат да връщат стойност чрез ключовата дума return. Това позволява на функцията да върне резултат след като бъде извикана. Например:
    function add(a, b) {
    return a + b;
    }
    var result = add(3, 5);

    console.log(result); // Очакван изход: 8
  • Анонимни функции: Анонимните функции са функции без име и се използват често като аргументи за други функции или при деклариране на променливи. Например:
    var greet = function(name) {
    console.log("Hello, " + name + "!");
    };

    greet("John");
  • Ламбда функции (arrow functions): Ламбда функциите са нова функционалност в ECMAScript 6 (ES6), която представлява по-кратък синтаксис за създаване на анонимни функции. Те се пишат чрез стрелката =>. Например:
    var greet = (name) => {
    console.log("Hello, " + name + "!");
    };

    greet("John");

Функциите в JavaScript са мощен инструмент за организиране на кода и могат да бъдат използвани за създаване на повторно използваеми блокове от логика. Те играят ключова роля в модерното програмиране и се използват широко във всички видове приложения.
