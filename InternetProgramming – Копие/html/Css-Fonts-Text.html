Стилизиране на текст и форматиране на шрифтове

Стилизацията и форматирането на текст е важна част от дизайна и създаването на уеб сайтове и апликации. Както вече знаем, при използване на HTML без CSS, големината и цвета на текста почти не могат да бъдат променяни с изключение на няколко тага.

Стилизацията на текст чрез CSS става след като селектираме даденият компонент от HTML документа и му добавим CSS "Надстройки"

Как се модифицира текст чрез CSS ?

Нека предположим че трябва да модифираме даденият параграф <p class="paragraph"> blah blah </p> , както виждаме тук имаме два варианта. Единият е да го селектираме директно по таг, докато другият вариант е да го селектираме чрез името на класа му. Вторият вариант е по-удачен, защото можем да таргетираме само съответният елемент без да модифицираме останалите параграфи в документа.

Промяна на големината на шрифта - .paragraph { font-weight: число в пиксели }

Промяна на цвета на текста - .paragraph { color: цвят }

Промяна на вида на шрифта - .paragraph { font-style: вид шрифт, italic например }

Местоположение на текста - .paragraph { text-align: място, center например }

Разстояние между символите - .paragraph { letter-spacing: число в пиксели }

Разстояние между редовете - .paragraph { line-height: число в пиксели }
