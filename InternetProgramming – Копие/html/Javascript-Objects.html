Обекти

Обектите в JavaScript са основен тип данни, който се използва за съхранение на данни във вид на ключ-стойностни двойки. Обектите могат да съдържат различни видове данни като примитивни типове данни (като числа, низове, булеви стойности), както и други обекти, масиви и дори функции. Ето основна информация за обектите в JavaScript:

  • Създаване на обекти: Обектите в JavaScript се създават чрез фигурни скоби {} и се състоят от нула или повече ключ-стойностни двойки, разделени със запетая. Например:
    var person = {
    name: "John",
    age: 30,
    city: "New York"
    };
  • Достъп до свойства: За да получите достъп до свойствата на обект, използвайте точка . след името на обекта, следвано от името на свойството. Например:
    console.log(person.name); // Очакван изход: "John"
    console.log(person.age); // Очакван изход: 30
  • Промяна на свойства: Свойствата на обектите могат да бъдат променяни, като присвоявате нови стойности на съществуващи свойства или добавяте нови свойства. Например:
    person.age = 35; // Промяна на стойността на съществуващо свойство
    person.gender = "male";
  • Изтриване на свойства: За да изтриете свойство от обект, използвайте оператора delete. Например:
    delete person.age;
  • Проучване на обекти: За да проучите свойствата на обекта, можете да използвате цикъл for...in. Например:
    for (var key in person) {
    console.log(key + ": " + person[key]);
    }
  • Вградени методи и свойства: Обектите в JavaScript имат вградени методи и свойства, като например Object.keys(), Object.values(), Object.entries(), които позволяват манипулирането на обектите по различни начини.

Обектите в JavaScript са мощен инструмент за организация на данните и логиката в програмите. Те се използват широко във всички видове приложения, от уеб разработка до създаване на backend системи и много други.
