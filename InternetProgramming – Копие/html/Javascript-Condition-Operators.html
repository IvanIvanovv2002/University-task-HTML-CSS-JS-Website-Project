Условни оператори

В JavaScript условните оператори се използват за вземане на решения в зависимост от различни условия. Те включват оператори като if, else if, else,switch-case и тернарни оператори. Ето какво представляват тези оператори и как се използват:

  • If оператор: операторът се използва, за да се изпълни блок от код, ако дадено условие е истина (true).
    if (условие) { // код, който се изпълнява, ако условието е истина }
  • else if оператор: операторът се използва, за да се провери допълнително условие, ако първото условие е лъжа (false).
    if (условие1) {
    // код, който се изпълнява, ако условие1 е истина
    } else if (условие2) { // код, който се изпълнява, ако условие2 е истина }
  • else оператор: операторът се използва, за да се изпълни блок от код, ако нито едно от предходните условия не е истина.
    if (условие1) {
    // код, който се изпълнява, ако условие1 е истина
    } else if (условие2) {
    // код, който се изпълнява, ако условие2 е истина
    } else { код в случай че нито едно от условията не е вярно. }
  • switch оператор: операторът се използва, за да се избере измежду няколко възможни блока от код, в зависимост от стойността на една променлива.
    switch (израз) {
    case стойност1:
    // код, който се изпълнява, ако изразът има стойност стойност1
    break;
    case стойност2:
    // код, който се изпълнява, ако изразът има стойност стойност2
    break;
    default: // код, който се изпълнява, ако нито една от предходните стойности не съвпада с израза
    }
  • тернарен оператор: Тернарният оператор е кратка форма за изразяване на условия. Той позволява да се избира между два израза в зависимост от стойността на дадено условие. Синтаксисът на тернарния оператор е: условие ? израз1 : израз2;
    var age = 20;
    var status = (age >= 18) ? "Пълнолетен" : "Непълнолетен";
    console.log(status); // Очакван изход: "Пълнолетен"
