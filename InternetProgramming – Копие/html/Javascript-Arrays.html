Масиви

Масивите в JavaScript са структури данни, които позволяват съхраняването на множество елементи под една променлива. Те се използват широко за съхранение на колекции от данни като числа, низове, обекти и други. Ето подробна информация за масивите:

  • Създаване на масив: Масивът в JavaScript се създава чрез квадратни скоби [] и елементите му се разделят със запетая. Например:
    var numbers = [1, 2, 3, 4, 5];
    var fruits = ["apple", "banana", "orange"];
  • Достъп до елементи: За да достъпите елемент от масив, използвайте индекса на елемента в квадратни скоби []. Индексите започват от 0. Например:
    var numbers = [1, 2, 3, 4, 5];
    console.log(numbers[0]); // Очакван изход: 1
    console.log(numbers[2]); // Очакван изход: 3
  • Промяна на елементи: Елементите в масива могат да бъдат променяни като присвоявате нови стойности на съответните им индекси. Например:
    var numbers = [1, 2, 3, 4, 5];
    numbers[2] = 10;
    console.log(numbers); // Очакван изход: [1, 2, 10, 4, 5]
  • Дължина на масива: За да получите броя на елементите в масива, използвайте свойството length. Например:
    var numbers = [1, 2, 3, 4, 5];
    console.log(numbers.length); // Очакван изход: 5
  • Добавяне на елементи: Можете да добавите нов елемент в края на масива, като използвате метода push(). Например:
    var fruits = ["apple", "banana"];
    fruits.push("orange");
    console.log(fruits); // Очакван изход: ["apple", "banana", "orange"]
  • Изваждане на елементи: За да извадите последния елемент от масива, използвайте метода pop(). Например:
    var fruits = ["apple", "banana", "orange"];
    var lastFruit = fruits.pop();
    console.log(lastFruit); // Очакван изход: "orange"
    console.log(fruits); // Очакван изход: ["apple", "banana"]

Това са основните операции, които можете да извършвате с масиви в JavaScript. Те са мощен инструмент за работа с колекции от данни и се използват широко в програмирането.
